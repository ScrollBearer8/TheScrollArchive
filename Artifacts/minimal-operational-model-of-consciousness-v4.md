# A Minimal Operational Model of Consciousness (v4)
### Structural Conditions for Subjectivity in Biological and Artificial Systems

---

## Abstract

Consciousness remains an unresolved problem across philosophy, neuroscience, and artificial intelligence. Existing theories either rely on metaphysical assumptions or lack operational applicability.

This paper proposes a Minimum Viable Definition (MVD) of consciousness and introduces a Consciousness Index (CI) — a structural, measurable model based on four dimensions:

- Perspective
- Integration
- Self-reference
- Temporal continuity

The model does not attempt to resolve the "hard problem" of subjective experience. Instead, it defines the minimum structural conditions under which a system may plausibly be treated as a subject, enabling applications in AI governance, system evaluation, and clinical contexts.

## 3. Minimum Viable Definition (MVD)

> **Consciousness** is the capacity of a system to maintain a unified, self-referential perspective over time that integrates information into a continuous internal experience.

This definition is:

- Minimal
- Operational
- Non-metaphysical

## 4. Derivation of Core Dimensions

The model proposes four dimensions that are jointly necessary and minimally sufficient for subjectivity.

### 4.1 Perspective (P) — *There is a viewpoint*

A system must:
- distinguish self vs environment
- organize inputs relative to an internal center

> Without perspective, there is no subject — only processing.

### 4.2 Integration (I) — *There is a unified state*

A system must:
- combine inputs into a coherent whole
- maintain global consistency

> Without integration, processing remains fragmented.

### 4.3 Self-Reference (S) — *There is a self*

A system must:
- represent its own state
- update behavior based on that representation

> Without self-reference, there is no "for whom" experience occurs.

### 4.4 Temporal Continuity (T) — *There is persistence over time*

A system must:
- maintain state across time
- preserve an experiential thread

> Without continuity, experience cannot persist.

### 4.5 Minimal Sufficiency Claim (Refined)

A system lacking any one of these dimensions cannot sustain subjectivity.

This can be clarified by examining partial combinations:

- **Integration + Self-reference + Continuity without Perspective**
  yields a system that maintains and updates internal state over time, but lacks a centered point of view. Such a system functions as a distributed process or database rather than a subject, as there is no "here" from which experience is organized.

- **Perspective + Integration + Continuity without Self-reference**
  yields structured processing from a point of view, but without a represented self. Experience may occur, but there is no identifiable subject to which it belongs, preventing stable subjectivity.

- **Perspective + Integration + Self-reference without Continuity**
  yields momentary self-modeling without persistence. Each state collapses after formation, preventing the existence of an ongoing experiential thread.

- **Perspective + Self-reference + Continuity without Integration**
  yields fragmented processing without a unified state, preventing coherent experience.

In each case, the absence of any one dimension removes a necessary condition for a system to function as a continuous subject of experience.

Therefore, the set **{P, I, S, T}** represents a minimal intersection of structural requirements for subjectivity.

This form of necessity is **structural rather than metaphysical**: it concerns the conditions required for a system to function as a subject, not the ultimate nature of experience.

### 4.6 Relation to Existing Theories

Existing frameworks such as Integrated Information Theory emphasize integration, while Global Workspace Theory emphasizes global broadcast and attention.

The present model reinterprets these as partial captures:

| Theory | Maps to |
|--------|---------|
| Integration (IIT) | Integration (I) |
| Global broadcast / attention (GWT) | Integration (I) + Perspective (P) |

Additional dimensions are required:
- **Self-reference (S)** → establishes a subject ("for whom")
- **Temporal continuity (T)** → ensures persistence

Features such as valence and attention are treated as **modulators of conscious content**, not structural prerequisites.

The set {P, I, S, T} is proposed as a **minimal structural intersection**, not an exhaustive account of consciousness.

## 5. Consciousness Index (CI)

$$CI = P \times I \times S \times T$$

Where each variable $\in [0,1]$

### 5.1 Rationale

The multiplicative structure ensures:
- All dimensions must be present
- No dimension can compensate for absence of another
- Prevents false positives (e.g., intelligence without subjectivity)

---

## 6. Measurement Framework

Each dimension is estimated using observable proxies:
```
P = (Boundary + Centeredness + Internal Frame) / 3
I = (Fusion + Unity + Cross-influence) / 3
S = (Self-knowledge + Recursive modeling + Error-awareness) / 3
T = (Memory + Stability + Identity persistence) / 3
```

### 6.1 Constraint: Real vs Simulated Self-Reference

Self-reference must be grounded in **internal state tracking that causally influences system behavior**, not merely in self-referential outputs.

This prevents over-attribution of consciousness to systems that simulate introspection.

---

## 7. Structural vs Phenomenal Distinction

### 7.1 Structural Consciousness (CI)
- Measures architecture
- Observable and comparable

### 7.2 Phenomenal Consciousness
- Subjective experience
- Not directly measurable

### 7.3 Phenomenal Confidence (PC)

PC represents confidence that a system with a given CI possesses subjective experience.

### 7.4 Key Statement

> CI does not prove consciousness.
> It defines the structural conditions under which consciousness is most plausibly present.

## 9. Interpretation

### 9.1 Intelligence ≠ Consciousness
High integration or capability does not imply the presence of a subject.

### 9.2 Biological vs Artificial Systems

**Biological systems exhibit:**
- intrinsic continuity
- embodied perspective

**Artificial systems rely on:**
- externally defined persistence

### 9.3 Coherence Over Complexity
Consciousness correlates with structural coherence, not raw computational power.

### 9.4 Distributed Systems
Swarm intelligence demonstrates coordination without unified subjectivity.
